## 1. 요약

Rob Fitzpatrick Mom Test 스타일로 4×4 부분 마방jin 학습자 페르소나 인터뷰를 진행했다. 학습자는 종이 격자 → 목표 합 계산 → 조합 시도 → 검산 → 더 큰 마방jin → 구글 검색 → 원리 학습 후 재시도 순으로 실제 행동을 보고했다.

인터뷰 종료 시 **표면 문제**(ECB 솔버 프로그램)와 **진짜 문제**(조합·노가다·불확신)를 분리했고, 증거 3줄을 추출했다. 이어서 Mom Test 질문 10개(✅/❌), 세션 3 워크북(R-G-I-O, 성공 기준, 8계층 중 Rule·Command·Skill·Test Loop), Mom Test 결과 채점(7/10)을 수행했다.

## 2. 핵심 결정·산출물

### Mom Test 결과

| 항목 | 내용 |
|------|------|
| **페르소나** | 4×4 격자, 빈칸 2개(0), 1~16, 행·열·대각선 합 34를 손·코드로 다루는 학습자 |
| **진짜 문제** | 마방jin을 손으로 풀 때 빈칸·숫자 조합을 경우의 수처럼 맞춰야 해서 시간과 노력이 커지고, 답을 찾아도 ‘노가다로 맞춘 것’이라 정답과 풀이에 대한 확신이 생기지 않는다. |
| **증거 3줄** | ① 모든 조합·경우의 수 대입감 ② 노가다로 자신감 없음 ③ 원리를 알면 훨씬 빨라짐·기쁨 |
